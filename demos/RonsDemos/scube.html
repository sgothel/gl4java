Scube applet
Originially written by David G Yu/Mark J. Kilgard/Silicon Graphics
Ported to Java by Ron Cemer
Click inside the window, then press any of the following keys:
L/l: toggle lighting
S/s: toggle SGI logo on/off
Q/q: toggle between quads and polygons for drawing
F/f: toggle fog
1: linear fog mode
2: exponential fog mode
3: exponential squared fog mode
p/P: toggle motion (pause/resume)
spacebar: single-step while paused
