Particle applet
Originially written by Nate Robins
Ported to Java by Ron Cemer
Drag the mouse inside the window to rotate the view.
Click inside the window, then press any of the following keys:
W/w: waterfall mode
F/f: fountain mode
S/s: toggle sphere collision
L/l: toggle between points lines
P/p: increase/decrease point size
+/-: increase/decrease number of particles
R/r: reset view rotation
