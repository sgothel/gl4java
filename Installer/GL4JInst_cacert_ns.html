Warning ! New certificate !
Before you can begin the GL4Java installation under Netscape, you must first must:
   - delete the old certificate and
   - re-import the new signing certificate.

To delete the old certificate, open the Netscape dialog:
   Communicator.Tools.Security
Then click on Certificates.Signers and delete the old GL4Java certificate !

The certificate enables the GL4Java installer to install the required GL4Java software components on your system.

Import the signing certificate, while using the link below.

A dialog box will pop up.

When the "Accept this Certificate Authority for Certifying software developers" box comes up:
    BE SURE THIS BOX CONTAINS A CHECK MARK.  FAILURE TO PUT A CHECK MARK IN
    THIS BOX WILL RESULT IN FAILURE TO INSTALL THE GL4JAVA LIBRARIES.

When the "Warn me before sending information to sites certified by this Certificate Authority" box comes up:
    BE SURE THIS BOX CONTAINS A CHECK MARK.  THIS IS AN EXTRA SECURITY PRECAUTION.

When asked for a name, enter the following name, exactly as shown: GL4Java

Click here to import the signing certificate. Valid from Tue Feb 13, 2001 to Sun May 13, 2001

After you have imported the certificate, Click here to begin the GL4Java installation.
