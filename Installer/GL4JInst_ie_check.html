For Win32 & InternetExplorer Users Only Be sure to have the MS-JavaVirtualMachine build 3186 or later installed !

If you have java enabled, please have a look in the applet field below !!

Otherwise check your Java Console (under the View menu) -
it should say something like
"Microsoft (R) VM for Java, 5.0 Release 5.0.0.3186".
If the last four digits are 3186 or above, you are fine.

Get the new JVM direct from the Microsoft-Webside, if necessary:

http://www.microsoft.com/java/vm/dl_vm40.htm.
http://www.microsoft.com/java/.
